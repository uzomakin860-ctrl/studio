# **App Name**: EchoCanvas

## Core Features:

- Speech Input: Accepts spoken words, turning them into text.
- Text Display: Showcases the text input from the user in a clean and readable way.
- Generative Response: Utilizes AI to craft an appropriate response based on the text input; decides if information needs to be added by consulting available tools.
- Response Display: Clearly displays the AI's response to the user.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to create a sense of calm reflection and deep thought.
- Background color: Very light lavender (#F0E6EF), a desaturated version of the primary color to provide a soft, non-distracting backdrop.
- Accent color: Deep purple (#9400D3) to complement the primary color, making interactive elements pop.
- Headline font: 'Space Grotesk' sans-serif for headlines and short amounts of body text, to give a tech-forward feel.
- Body font: 'Inter' sans-serif for longer text to give the site a modern and objective feel.
- Simple, outlined icons to represent different functionalities and keep a clean look.
- Clean, centered layout with ample spacing for readability.
- Smooth transitions and subtle animations to improve user experience.
